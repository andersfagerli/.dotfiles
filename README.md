## 1. Nord GNOME Terminal

```
git clone https://github.com/arcticicestudio/nord-gnome-terminal.git
cd nord-gnome-terminal/src
./nord.sh
```

In a terminal, go to *Edit*->*Preferences*->*Profiles* and choose Nord as default

## 2. GTK-font to System San Francisco Display

```
sudo apt install lxappearance
```
Open *lxappearance* from *dmenu* and choose *SFNS Display* as default font. Choose *Regular* as style and 13 as font size.

## 3. Node.js for CoC
```
sudo snap install node --classic
```
